Eve brings transcript-first spoken-word video editing into Emacs. Instead of
scrubbing a timeline to trim interviews, podcasts, lessons, or talking-head
recordings, you edit the words that were said. Delete a phrase, move a
paragraph, add a section marker, attach b-roll, and compile the result without
leaving the editor.

It stays fast enough for iterative work: playback and preview live inside the
editing workflow, draft compiles reuse cached and pre-rendered segments, and a
small text change usually means only the affected parts are rendered again.

The full manual lives in [[https://github.com/behaghel/eve.el/blob/main/docs/eve.org][docs/eve.org]].

* Getting Started

Install the Emacs package:

#+begin_src emacs-lisp
(use-package eve
  :straight (:type git :host github :repo "behaghel/eve.el"))
#+end_src

For standard Emacs use, =M-x eve-transcribe= first looks for =eve= on your
=PATH=. If it does not find one and =eve-cli-program= is still at its default,
Eve bootstraps the bundled CLI helper from this checkout the first time you
need it.

Then use Eve for the main workflow:

1. Run =M-x eve-transcribe= on a directory of recordings.
2. Eve keeps a visible running status while transcription is in progress, then
   opens the generated =.tjm.json= manifest automatically in =eve-mode=.
3. Delete words, reorder segments, add markers, or attach b-roll.
4. Press =C-c SPC= to play from point and =C-c C-c= to compile a draft.
5. Press =C-u C-c C-c= for a final H.264 export.

Opening any =*.tjm.json= file activates =eve-mode= automatically. The manual in
[[https://github.com/behaghel/eve.el/blob/main/docs/eve.org][docs/eve.org]] covers installation details, commands, and editing features in
full.

* Why Eve

- Transcript-first editing for spoken-word video inside Emacs.
- Edit by deleting or reordering text instead of trimming a timeline.
- Preview and playback stay close to the text you are editing.
- Draft renders are fast because changed segments are cached and pre-rendered.
- Final delivery uses the same workflow, with =C-u C-c C-c= for H.264 export.

* Installation Notes

- Emacs 29.1 or later.
- =mpv= for playback.
- Python 3.12 or later for the CLI.
- =hydra= is optional and enables the =?= cheatsheet popup.
